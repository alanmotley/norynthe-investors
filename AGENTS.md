# AGENTS.md

This repository hosts the private investor-facing Norynthe site.

Priorities:
- Preserve Norynthe brand language and visual polish.
- Do not change core messaging unless asked.
- Keep the site lightly access-controlled.
- Prevent public indexing by search engines.
- Add or preserve robots meta tags:
  - noindex
  - nofollow
  - noarchive
  - noimageindex
- Add or preserve robots.txt rules blocking crawlers.
- Do not expose sensitive investor content in public navigation.
- Prefer simple static HTML/CSS/JS unless a framework already exists.

When making changes:
- Create focused commits.
- Explain what changed.
- Flag any security limitation clearly.
